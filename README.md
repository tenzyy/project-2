# Pong Game

A basic pong game using SVGs.
Background image: gif file added to board.js


## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* a: up, music (not on loop)
* z: down

**Player 2:**
* ▲ : up
* ▼: down
